## 주요 기능
* **플랫폼 통합 랭킹**: 두 방송 플랫폼의 스트리머의 실시간 시청자 수를 비교해 1위에서 48위까지를 한 곳에서 볼 수 있습니다.
* **통합 검색**: 여러 플랫폼을 이동할 필요 없이, 단 한 번의 검색으로 양쪽 플랫폼의 스트리머 검색을 할 수 있습니다.
* **스트리머 팔로우**: 플랫폼 구분 없이 좋아하는 스트리머 팔로우를 해 한 곳에서 볼 수 있습니다.
* **실시간 방송 알림**: 팔로우한 스트리머는 방송을 시작할 때 알림이 옵니다.
* **백그라운드 알림**: 앱을 최소화해도 시스템 트레이에서 백그라운드로 방송 상태를 확인해 알림을 보내줍니다.
* **완전 무료**: 네. 광고도 결제도 아무것도 받지 않습니다. 그저 즐기세요!

## 개발 상태
 - 필수적인 기능은 모두 구현이 끝났습니다.
 - 이후론 간단한 최적화같은 작업만 하면 됩니다
 - 물론 신기능 추가도 염두해두고 있습니다

## 개발
* **언어**: C#
* **Framework**: .NET 8.0 / WPF
* **대상 OS**: Windows 10/11 (그 밑은 작동하지 않습니다)
* **개발자**: THESALT STUDIO

## 참고사항
 - 이 프로그램은 인터넷 방송 플랫폼인 치지직과 SOOP 데이터만을 기반으로 작동합니다
 - 이 프로그램은 두 방송 플랫폼의 공식 프로그램이 아닙니다.
 - 실제 방송 시청은 각각의 사이트에서 진행됩니다. 이 프로그램은 오직 일종의 허브 역할만 합니다.
